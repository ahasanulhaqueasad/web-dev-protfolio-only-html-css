# 🚀 Web Developer Portfolio

A modern, responsive web developer portfolio website built from scratch. This project showcases my skills, experience, and projects in a clean, minimalist, and user-friendly interface.

## 🛠 Technologies Used

* **HTML5:** Semantic structure and content organization.
* **CSS3:** Custom styling, layout management, and responsive design.
* **Google Fonts:** Utilizing "Open Sans" for a clean and professional look.

## ✨ Key Features

* **Fully Responsive:** Optimized for mobile, tablet, and desktop screens.
* **Clean UI:** Modern design with premium aesthetics.
* **Interactive UX:** Smooth hover transitions and clickable elements for better user engagement.
* **Contact Integration:** A custom, functional contact section to easily connect with me.
* **Experience & Skills:** Dedicated sections to highlight my technical expertise and background.
